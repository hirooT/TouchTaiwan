電子載板3D+2D智慧視覺檢測模組
3D+2D intelligent visual inspection module
自主3D+2D視覺感測模組，快速重建待測工件的2D色彩與3D形貌，以3D+2D自動特徵精密量測技術，結合2D快速定位特徵與高精度3D量測，解決既有3D產品速度不足瓶頸，提供高效穩定的自動精密量測，提升檢測效率與品質，已導入電子載板達到零漏檢，未來可擴散應用至LED、被動元件及IC封裝等檢測應用。
The domestic intelligent visual sensing module is developed to reconstruct the object’s 2D color and 3D shape. Intelligent 3D+2D feature recognition and measurement technology has the advantage of fast 2D positioning and high accuracy 3D measuring. High efficiency and quality of automatic inspection can be realized using the proposed intelligent module. This module has been implemented in IC substrate production line and the goal of zero missing detection is achieved. In the future, the 3D+2D intelligent visual inspection module can be also applied in LED, passive components and IC package.
